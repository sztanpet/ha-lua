# ha-lua add-on repository

A Home Assistant add-on repository. Add this URL under
**Settings → Add-ons → Add-on store → ⋮ → Repositories**:

```
https://github.com/sztanpet/ha-lua
```

## Add-ons

| Add-on | Description |
|--------|-------------|
| [**ha-lua**](./ha-lua) | Lua scripting engine for Home Assistant — mirrors entity state into SQLite and runs automations as per-script Lua VMs. |

See [`ha-lua/README.md`](./ha-lua/README.md) for the add-on's documentation and
development guide.
